# marketing-service

营销服务,核心职责是促销转化, 包含功能模块：兑换码、优惠券、秒杀、积分、秒杀、拼团等通用权益功能

## 通用服务化

- 兑换码、优惠券、积分等通用权益功能
- 采用插件化设计：兑换码作为核心插件，支持多产品线配置
- 功能增强
  - 多类型支持（一次性/多次/比例折扣等）
  - 组合验证规则（时间/用户层级/产品范围）
  - 批量生成+异步发放能力

## 中台化演进阶段
